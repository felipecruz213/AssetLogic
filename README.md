# AssetLogic
AssetLogic: CRUD simples de uma gestão de inventário com PHP, MySQL e Bootstrap.
---

O projeto é uma site simples, onde pode adcionar, apagar, editar itens, com categoria de hardware, software e consumíveis, com preço e quantidade

---

Desenvolvi no Manjaro Linux, com o servidor apache em um container do Docker, com php 8.2, e MySQL Server
